# Image Assets for Hide-And-Seek Game Project

## Required Images

To display the Hide-And-Seek game screenshots on the project page, please add the following images to this directory (`assets/images/`):

1. **hide-seek-menu.jpg** - Main menu screenshot showing the game menu with "Ready To Hide?", "Ready To Seek?", "Settings", and "Quit" options
2. **hide-seek-gameplay.jpg** - Gameplay screenshot showing the player character in the game world
3. **hide-seek-gameover.jpg** - Game over screen showing "GAME OVER" with retry and quit options
4. **hide-seek-scene.jpg** - Another scene/gameplay screenshot showing the game environment

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080 (16:9 aspect ratio) or similar
- **File Names**: Use the exact names listed above

## How to Add Images

1. Save your screenshots with the exact filenames listed above
2. Place them in the `assets/images/` directory
3. The project page will automatically display them in the gallery section

## Alternative

If you don't have the images yet, the page will show placeholder boxes. You can add the images later.
