Requirements

To run this game, you will need:

  • Windows OS - Windows 10 or later recommended
  • DirectX 11 or higher - For graphics rendering
  • Graphics Card - Minimum specs should meet Unreal Engine requirements
  • Sufficient Storage Space - For game files and assets

Installation

1. Downloading the Game Setup

Download the game files from the Google Drive link provided above. The files have been uploaded there due to size limitations of GitHub.

2. Installing the Game Files

After downloading, follow these steps:

  1. Extract the downloaded files to your desired location
  2. Locate the extracted folder and navigate to FinalYear/Content/Paks
  3. Place the downloaded .pak file in this folder

Note: Make sure the file structure matches: FinalYear/Content/Paks/YourPakFile.pak

3. Running the Game

Once the installation is complete:

  1. Navigate to the game directory
  2. Look for the HideAndSeek.exe (or equivalent executable file)
  3. Double-click to run the game

How to Play

Once the game is running:

Starting the Game

Choose the number of players and whether you'd like to play as a hider or a seeker.

Objective

Hiders

Find a good hiding spot and avoid being caught by the seekers. Use the environment to your advantage!

Seekers

Locate the hiders within the given time limit. Use your wits and observation skills to find them!
